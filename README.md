🏥 Chronic Care Manager

🚀 A smart healthcare management system designed to monitor, manage, and improve chronic patient care efficiently.

📌 Features
👩‍⚕️ Patient Management System
📊 Health Monitoring Dashboard
💊 Medication Tracking
📅 Appointment Scheduling
🔐 Secure Data Handling
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript / React
Backend: Node.js / Python
Database: MongoDB / MySQL
⚙️ Installation
1️⃣ Clone the repository
git clone https://github.com/1112-neha/FSAD_Hackthon_chronic-care-manager.git
2️⃣ Navigate to project folder
cd FSAD_Hackthon_chronic-care-manager
3️⃣ Install dependencies
npm install
4️⃣ Run the project
npm run dev
🎯 Usage
Open browser and go to: http://localhost:5173
Start managing patient records and healthcare data
